# Basic Syntax

## 변수

### 식별자(변수명) 작성 규칙
- 반드시 문자, 달러 또는 밑줄로 시작
- 대소문자를 구분
- 예약어 사용 불가
- 카멜 케이스 : 변수, 객체, 함수에 이용
- 파스칼 케이스 : 클래스, 생성자에 사용
- 대문자 스네이크 케이스 : 상수에 사용

### 변수 선언 키워드
1. let
  - 블록 스코프를 갖는 지역 변수를 선언
  - 재할당 가능
  - 재선언 불가능
  - ES6에서 추가
2. const
  - 블록 스코프를 갖는 지역 변수를 선언
  - 재할당 불가능
  - 재선언 불가능
  - ES6에서 추가
3. var

- 원시 자료형
  - Number

  - undefined : 변수 선언 이후 직접 값을 할당하지 않으면 자동으로 할당됨
  - boolean
